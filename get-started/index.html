Get started with...

Dandelion-Core

First it might be a good idea to familiarize yourself with the key concepts of Dandelion-Core.

For example:

  • What is an asset bundle
  • How to use an asset locator

The next step would be to give a try to the Dandelion sample applications.

You can also give a try to the Dandelion version of the Spring PetClinic application. In this version, all assets are based on Dandelion, so if you are already familiar with Spring and this sample application, this is a really good starting point.

Dandelion-Datatables

Dandelion-Datatables is the first and most advanced component of the framework focused on HTML tables.

Lots of samples are available, each of them focused on a particular feature or a set of related features. Feel free to browse/clone/fork them!
