Basics of styling

There are a number of ways in which DataTables allows you to style tables and HTML it adds to the document. Dandelion-DataTables supports some of them but also provides themes integration such as the Bootstrap2 theme. But let's see the basics first.

Styling the table

Some attributes are available in the JSP taglib, allowing you to style different elements:

  • the cssStyle table attribute allows you to apply inlined style. It's a pass through attribute for the style HTML attribute
  • the cssClass table attribute allows you to apply some CSS classes to the table. It's a pass through attribute for the class HTML attribute
  • finally, the cssStripes table attribute allows you to apply a sequence of CSS classes that should be applied to displayed rows. This sequence of CSS classes may be of any length, and DataTables will apply each class sequentially, looping when required

Using JSP

You will find below an example that shows how to use the cssClass table attribute and the cssClass and cssStyle column attributes. 

<datatables:table id="myTableId" data="${persons}" cssClass="myClass">
   <datatables:column title="Id" property="id" />
   <datatables:column title="FirstName" property="firstName" />
   <datatables:column title="LastName" property="lastName" />
   <datatables:column title="Mail" property="mail" />
</datatables:table>

Using Thymeleaf

Since the Thymeleaf syntax is based on pure HTML, most of the previous attributes are natively available in HTML.

However, if for example you need to set the stripe classes, you can use the dt:stripeclasses table attribute. 

<table id="myTableId" dt:table="true" dt:stripeclasses="class1,class2">
   <thead>
      <tr>
         <th>Id</th>
         <th>Firstname</th>
         <th>Lastname</th>
         <th>Mail</th>
      </tr>
   </thead>
   ...
</table>

Styling the columns

As for the table, some attributes are available in the JSP taglib:

  • the cssStyle and cssCellStyle column attributes, allowing you to apply inlined style respectively on the corresponding <th> tag and all <td> tags of the same column
  • the cssClass and cssCellClass column attributes, allowing you to apply CSS classes on the corresponding <th> tag and all <td> tags of the same column

Using JSP

You will find below an example that shows how to use the cssClass table attribute and the cssClass and cssStyle column attributes. 

<datatables:table id="myTableId" data="${persons}">
   <datatables:column title="Id" property="id" cssClass="myColumn1Class" />
   <datatables:column title="FirstName" property="firstName" cssClass="myColumn2Class" />
   <datatables:column title="LastName" property="lastName" cssStyle="text-align:center;" />
   <datatables:column title="Mail" property="mail" />
</datatables:table>

Using Thymeleaf

Since the Thymeleaf syntax is based on pure HTML, most of the previous attributes are natively available in HTML.

However, if for example you need to set the stripe classes, you can use the dt:stripeclasses table attribute. 

<table id="myTableId" dt:table="true" dt:stripeclasses="class1,class2">
   <thead>
      <tr>
         <th class="myColumn1Class">Id</th>
         <th class="myColumn2Class">Firstname</th>
         <th style="text-align:center;">Lastname</th>
         <th>Mail</th>
      </tr>
   </thead>
   <tbody>
      <tr th:each="person : ${persons}">
         <td th:text="${person?.id}">1</td>
         <td th:text="${person?.firstName}">John</td>
         <td th:text="${person?.lastName}">Doe</td>
         <td><a th:href="${'mailto:' + person.mail}" th:text="${person.mail}">john@doe.com</a></td>
      </tr>
   </tbody>
</table>

Using external styles

DataTables will add a number of elements to the page, each with a suitable default ID and/or class to allow CSS selectors to address the element.

You will find below the reference of the id's and classes applied to HTML elements added by DataTables and Dandelion-Datatables:
